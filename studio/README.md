# Hearthlight Studio

Not started yet. Per the roadmap in `hearthlight-gdd.md` (Part IX), Studio work
begins at Phase 4, after Map/VN/Battle modes exist in `/engine` to edit content
for. It will follow the same scaffold pattern as the engine: custom elements
(`<app-layout>`, `<app-tabs>`, `<app-modal>`), Shadow DOM, and one event bus —
no framework.
